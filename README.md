# rebar.bot

An interactive landing page featuring the Rebar bot mark. The robot tracks your
cursor, blinks, wanders when idle, and boops on click before sending you to
[rebar.is](https://rebar.is).

Single self-contained `index.html` — no build step. Open it directly or serve
the folder statically.

## Files

- `index.html` — the page (inline SVG bot + CSS + vanilla JS)
- `logo-bot.svg` — the Rebar bot mark, vectorized from the source PNG

Dark theme, Rebar brand palette (`#101820` / teal `#40a6ab` / red `#df2927`),
Saira typeface.
